Tic Tac Toe

An application to illustrate the Minmax algorithm

To locate the optimal move, the program use the Minmax algorithm.

Additionally, alpha-beta pruning is included to enhance the Minmax algorithm.

Currently the web version is unavaliable

If you wanna try it on your machine

  • Clone the Git Hub repository

    $git clone https://github.com/kyawzinmyat/tictactoe

  • Install pygame

    $pip3 install pygame

    Or

    $pip3 install -r requirements.txt

  • Run runner.py file

    $python3 runner.py

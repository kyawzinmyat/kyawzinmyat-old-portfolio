Maze generating program that uses randomized DFS(depth first search)

A program to illustrate how searching algorithms including

  • Depth First Search
  • Breadth First Search
  • Greedy Best First Search
  • A* Search

Find a path and how different algorithms yield the results (the path)

Currently the web version is unavaliable

If you wanna try it on your machine

  • Clone the Git Hub repository

    $git clone https://github.com/kyawzinmyat/mazeGen

  • Install pygame

    $pip3 install pygame

  • Run game.py file

    $python3 game.py

    • Here is a video of how its work!

    Conclusion

    You can compare how different algorithms produce the path by checking the path length

    You can also differentiate how DFS and BFS (uninformed search) take steps to get the path

    You can also see how Greedy Best First Search yields a path but not the best or optimized one

    Also the length of the path produced from BFS and A* search
